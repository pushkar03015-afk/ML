# Customer Churn Prediction

## Project Overview

This project focuses on predicting customer churn for a subscription-based service using Machine Learning. The goal is to identify high-risk customers early by analyzing demographic information, behavioral patterns, and financial activity.

---

#  Key Outcomes

- Built a **Random Forest Classifier** achieving **97% accuracy**
- Designed a robust ML pipeline with preprocessing and hyperparameter tuning
- Generated actionable business recommendations for churn reduction

---

#  Key Business Insights

##  1. Customer Support as a Churn Trigger

Customers with **3+ support calls** showed a significantly higher probability of churn, indicating service friction as a major churn driver.

###  Recommendation
Implement an automated **High-Touch Support System** for customers crossing the support-call threshold.

---

##  2. Price Sensitivity & Spending Patterns

Higher churn rates were observed among customers with lower long-term value perception.

###  Recommendation
Introduce:
- Loyalty rewards
- Long-term membership discounts
- Premium retention offers
